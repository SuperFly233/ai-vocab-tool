# Changelog

## v0.9.10 - 2026-04-28

- 收藏历史条目改为更精致的金色高亮、细光边和柔和背景，不再使用粗糙的单边色块样式。
- 修复历史筛选菜单点击选项后被全局点击处理立即关闭的问题，多选时菜单会保持展开。

## v0.9.9 - 2026-04-28

- API 配置弹窗底部按钮改为显式事件绑定，修复“保存配置”点击后没有反应的问题。
- API Key 输入框改为非密码字段，并关闭自动填充/密码管理识别，减少浏览器反复提示保存密码。
- 登录邮箱和登录密码字段补充标准 `autocomplete`，让浏览器更容易区分账号密码和 API 密钥。

## v0.9.8 - 2026-04-28

- API 配置外层只保留当前配置选择，新增和编辑改为弹窗填写名称、API URL、API Key 和 Model。
- 删除配置与清空全部配置逻辑重新整理，最后一组配置也能清回默认空配置。
- 新增模型列表查询代理 `/api/models`，可在配置弹窗里尝试读取 OpenAI-compatible 模型列表。
- 历史筛选改成自制多选菜单，每个分类都支持单选或多选，默认状态用短横显示。

## v0.9.7 - 2026-04-28

- 自定义 API 配置组不再使用原生选择器，改为当前配置卡片和自制切换菜单。
- 新增、保存、删除和恢复默认拆成独立按钮，避免把危险操作混在配置选择里。
- 配置列表会显示每组是否完整，以及当前模型或环境默认模型状态。

## v0.9.6 - 2026-04-28

- 历史筛选改为紧凑下拉控件，避免多选列表占用过大的垂直空间。
- 排序按钮收紧为工具条样式，减少历史页顶部拥挤和错位。
- 收藏历史卡片去掉右侧大星水印，改为轻量左侧高亮和按钮状态，避免遮挡操作区。

## v0.9.5 - 2026-04-28

- 修复重复触发同一条 toast 时，底部进度条不重新开始、自动关闭剩余时长变短的问题。
- 手动关闭 toast 时会同步清理对应计时器，避免后续再次打开时继承旧状态。
- 查询加载态移除圆形 spinner，只保留直线进度条，并优化等待卡片的背景、边框和高光效果。

## v0.9.4 - 2026-04-27

- 设置页移除“当前来源 / 环境变量 / 管理员限制”状态卡，减少对网页配置流程的干扰。
- API 配置组改为集中操作：新增配置、保存修改、删除当前配置。
- “恢复默认”移入 API 配置下拉菜单，让模型/API 管理入口更集中。

## v0.9.3 - 2026-04-27

- 历史筛选改为多选，语言、方向、词性、语体都支持同时选择多个条件。
- 对旧历史中的语言、方向、词性、语体字段做前端归一化，减少 `EN/en/英文`、`英译中/译为中文`、`v./动词` 这类重复筛选项。
- 更新 AI 输出 prompt，要求新生成结果使用固定枚举字段。
- 排序按钮文字和箭头保持居中，筛选控件在桌面和手机端宽度更统一。
- 清空历史的二次确认文案改为更明确的危险操作提示。

## v0.9.2 - 2026-04-27

- 在全部历史视图中，已收藏条目会显示更醒目的收藏卡片背景。
- 收藏条目新增左侧高亮条、轻微暖色光影和淡星标水印。
- 收藏状态也会显示在历史时间信息中，便于快速扫视。

## v0.9.1 - 2026-04-27

- 左上角品牌区新增可点击的网页图标，点击后回到首页。
- 进入主界面、点击品牌图标、点击首页按钮时会自动聚焦搜索框。
- 历史记录重新生成跳回首页时也会自动把焦点放回输入框。

## v0.9.0 - 2026-04-27

- 设置页新增多组 API 配置，可保存多套 API URL、API Key 和 Model。
- 支持在多组配置之间切换当前使用项，查询和追问会使用当前配置组。
- 旧版单组 API 设置会自动迁移为默认配置组。
- 云同步会合并不同设备上的 API 配置组，减少新设备拿不到接口设置的问题。

## v0.8.1 - 2026-04-27

- 右上角通知弹窗新增关闭按钮，自动消失前也可以手动关闭。
- 重做通知弹窗视觉样式，加入状态图标、侧边光条、扫光、进度条和更顺滑的进出动效。
- 通知弹窗改为单条可交互，不影响页面其它区域点击。

## v0.8.0 - 2026-04-27

- 修复云端同步流程，登录后自动合并本机和云端数据，不再依赖弹窗选择。
- 同步范围扩展到历史、收藏、API 设置、主题、布局和运行日志。
- 页面聚焦、从后台回到前台、以及登录状态下定时轮询时会静默拉取云端更新。
- 新增 `PROJECT_CONTEXT.md`，用于新对话快速衔接项目状态、同步策略和后续工作规则。
- README 增加同步策略和项目上下文维护要求。

## v0.7.0 - 2026-04-25

- 新增结果追问功能，可基于当前词条继续向 AI 提问。
- 追问回答以对话卡片显示在结果页下方。
- 追问会保存到对应历史记录中，历史详情里也可以继续查看。
- 追问支持卡片内编辑和删除。

## v0.6.0 - 2026-04-25

- 首页输入框取消伸缩特效，默认完整展示输入条、辅助输入和操作按钮。
- 支持在首页输入框按 Enter 快速查询，Shift+Enter 保留换行空间给后续扩展。
- 查询中显示请求阶段、接口来源和加载动画，减少等待时的黑箱感。
- 历史记录整块可点击进入详情，删除按钮仍保持独立操作。
- 历史详情支持排版、分区 JSON、编辑保存三种视图。
- 结果排版重做层级：义项序号靠左，词性单列，重点内容使用荧光笔底色。
- JSON 视图按 meta、headword、senses、collocations、register、confusions 分区展示。
- 降低全站字重密度，让标题、标签、正文和重点内容有更明确主次。

## v0.5.0 - 2026-04-25

- 新增自定义 API URL、API Key、Model 设置。
- 新增恢复接口默认、恢复出厂设置、运行日志和关于页。
- 历史清空按钮改为垃圾桶图标。

## v0.4.0 - 2026-04-25

- 增强历史记录搜索和排序。
- 优化结果卡片与 JSON 查看体验。
- 加入云端设置同步。
