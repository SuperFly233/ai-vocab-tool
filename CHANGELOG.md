# Changelog

## v0.7.0 - 2026-04-25

- 新增结果追问功能，可基于当前词条继续向 AI 提问。
- 追问回答以对话卡片显示在结果页下方。
- 追问会保存到对应历史记录中，历史详情里也可以继续查看。
- 追问支持卡片内编辑和删除。

## v0.6.0 - 2026-04-25

- 首页输入框取消伸缩特效，默认完整展示输入条、辅助输入和操作按钮。
- 支持在首页输入框按 Enter 快速查询，Shift+Enter 保留换行空间给后续扩展。
- 查询中显示请求阶段、接口来源和加载动画，减少等待时的黑箱感。
- 历史记录整块可点击进入详情，删除按钮仍保持独立操作。
- 历史详情支持排版、分区 JSON、编辑保存三种视图。
- 结果排版重做层级：义项序号靠左，词性单列，重点内容使用荧光笔底色。
- JSON 视图按 meta、headword、senses、collocations、register、confusions 分区展示。
- 降低全站字重密度，让标题、标签、正文和重点内容有更明确主次。

## v0.5.0 - 2026-04-25

- 新增自定义 API URL、API Key、Model 设置。
- 新增恢复接口默认、恢复出厂设置、运行日志和关于页。
- 历史清空按钮改为垃圾桶图标。

## v0.4.0 - 2026-04-25

- 增强历史记录搜索和排序。
- 优化结果卡片与 JSON 查看体验。
- 加入云端设置同步。
